    J1 Carousel Module

    J1 CAROUSEL Module is based on OWL Carousel V1 in the latest version of 1.3.3. OWL Carousel is a clean and neat jQuery slider plugin for creating fully responsive and touch-enabled carousel sliders.

    OWL Carousel V1 is not longer available on the Internet, for unknown reason. Anyway, J1 Template is using this version of OWL Carousel because the software does an excellent job, is based on a MIT license with no issues using them for private and business use. And offers a lot of great features!

    Simple Text Show

    A slider or carousel typically is used for images but the implementation for J1 Template of OWL Carousel support a lot more sources to be displayed as a slide show: simple text for example.

    Important statements or topics can be placed e.g. on top of an article or a paragraph to give them a better visibility. In one line, you can present a bunch of facts to know animated for the readers attention within as single line - no much space is needed.

    Parallax Text Show

    A more eye-minded type of a text show is a parallax text slider. If you want to place emphasis on you personal statements focussing the meaning, this type of a slide show may be interesting. Image based slide shows may draw off the readers attention from the text, therefor a pure text based presentation may the better choice.

    Parallax text shows are can be placed as headers or banners on a page. A lot of different animations can be used. Internally, J1 Template is using some of the really great CSS styles offered by annimate.css.

    Have a look a Dan Eden’s home page to see all the possible dynamic styles you can create based on pure CSS. Some of them are implemented for OWL Carousel for animation. See the documetation for the CAROUSEL Module for more details.

    Carousels and sliders are most used for image based data to animate images in a row. Find with the following some examples how to use a carousel for your picures.

    Slider + Caption + Lightbox

    Sliders can be used for a quite compact form of image galleries. This example show some pictures having individual caption text and support a lightbox to enlarge images full size. For the example below, a quite simple lightbox is used: Lightbox2.

    J1 Gallery Module

    To create image and video galleries, J1 Template implements the jQuery plugin Justified Gallery as the main gallery module. Beside Justified Gallery, a gallery based on pure Bootstrap code is available with Bootstrap Gallery.

    Justified Gallery is a great jQuery plugin to create responsive, infinite and high quality justified image galleries. J1 Template combines the gallery with lightboxes to enlage the images of a gallery.

    The same to Bootstrap Gallery. Bootstrap Gallery is a slim jQuery plugin to create galleries using the grid system of Bootstrap. As mentioned ealier, the gallery is simple but a good choice for even sized images or for video previews. Video previews are based typically on even sized thumbnail images. For that reason, a Bootstrap Gallery os a good choice.

    See both of them in action - and for sure all what you see is even responsive. Change the size of your current browser window, by width or height, to see what will happen!

    Image Galleries

    A bunch of images to display needs an elegant view to see all images from a preview. That’s what the J1 Gallery module is doing.

    Bootstrap Gallery

    Bootstrap Gallery is a simple gallery using Bootstrap’s grid system to order images grid based into rows and colums. Beside the simplicity, Bootstrap Gallery has some nice features you can turn on or off to give a gallery a special style and behaviour.

    As for all galleries, a lightbox is put on all images to enlarge and browse them for convenience and best access.

    Justified Gallery

    Pictures you’ve made are typically not even in size. Images may have the same size (resolution), but some of the are orientated landscape other may portrait. For that reason, a more powerful gallery is needed to create so-called masonry styled views.

    Justified Gallery is using a masonry grid layout. It works by placing elements in optimal position based on available vertical space, sort of like a mason fitting stones in a wall. You’ve probably seen it in use all over the Internet!

    HTML5 Video Galleries

    Playing video in a web page seems not that special since you can already watch videos on a web page through plugins like Flash Player, Quicktime or Silverlight. HTML5 video is the new standard way to show video on the web, instead of the previous de facto standard of using proprietary software like Flash or Quicktime.

    HTML5 video open the doors to your way presenting video content. Modern browsers support the video tag <video> for HTML5 video standard. Browsers have a build-in multimedia framework already for decoding and displaying video content. No need to use such proprietary software components!

    HTML5 Video support is given by Bootstrap Gallery combined with the Light Gallery lightbox.

    Two types of video sources are supported:

    1. videos from local files (your web space)

    2. videos from online sources (the Internet) like YouTube, Vimeo, etc.

    The combination of Bootstrap Gallery and the lightbox of LightGallery is quite usable for all the video content you want to present on your site. Galleries can placed elsewhere. You can use them for video blogs for example by placing them into your blog articles content or headers.

    Light Gallery can be used for free for private use but for business use, a commercial license is needed. See Light Gallery license how to use Light Gallery for commercial webs and projects.

    Local video content

    Digital image content, simple pictures or vidoes, are quite easy to make. Today, each and every mobile has a camera - not that bad! Presenting a bunch of (digital) pictures is very easy by using Justified Gallery for example. Videos created by a digi cam or a mobile can played by J1 Template using the HTML5 video support of Light Gallery.

    Two players are available with Light Gallery:

    1. a internal player used by default

    2. video.js, a excellent Javascript video library

    The HTML5 specification does not define which video and audio formats browsers should support. J1 LightGallery supports all typical types of HTML5 video formats browsers support today such as MP4, WebM, and Ogg.
